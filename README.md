## How to run this game:
1. Make sure you have Python installed.
2. Install pygame by running: pip install pygame
3. Go to the Folder "dist" and run Dodge.exe
4. Enjoy!

## Features:
- Player movement
- Theme song
- Simple graphics
- Easy controls includes crouching and jumping!
